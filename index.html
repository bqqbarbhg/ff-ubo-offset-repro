The demo renders two triangles over a red background. The first green triangle has UBO bound always at offset 0. The UBO offset of the blue triangle is configurable via the input above. According to my testing using Firefox on Windows+Nvidia the triangle is rendered white without error if the offset is greater than or equal to 64kB (0x10000). The backing uniform buffer is 1MiB (0x100000) in size and using an offset above that fails as it should.
